Lesson 1

  • CSS (Cascading Style Sheets) is a language used to style web pages, separating content from design, making them easier to maintain and more attractive.
  • It separates content from design, improving accessibility and responsiveness.
  • A webpage has three layers: content, presentation, and behavior.
  • The content layer defines and layouts text, images, animations, sounds, and videos using tags.
  • The presentation layer defines how people see the page in a browser, ensuring text can be zoomed without making the site unusable.
  • The behavior layer ensures real-time interaction with the web page, making it easy to use. CSS offers benefits such as time savings, easy maintenance, and visual appeal. It also allows for faster downloads, faster page loading, multiple link tags, better design flexibility, and easier maintenance and updating.

Lesson 2

CSS is a set of instructions for web browsers to display various elements on a webpage. It consists of three types: External Style Sheets, Embedded Style Sheets, and Inline Style Sheets.

  1. External Style Sheets are the most global and allow for consistent styles across pages.
  2. Embedded Style Sheets are used for creating document-wide style rules and are placed within an HTML document in the head tag.
  3. Inline Style Sheets are used for isolated changes to a headline, paragraph, picture, or other element.
CSS code should always be enclosed in HTML comments to prevent browsers from seeing the CSS code.
The style attribute is used to define the style for a single element within an HTML document.
  • Inline styles are placed within the head part of the HTML file
  • Embedded styles are defined by the container tag style and placed within the HTML file. Embedded styles are document-wide styles, affecting all paragraphs in the web page.
Linked style sheets, or linked styles, are made outside the HTML file and are linked via the link tag. These styles do not contain any HTML code but only style rules.
A CSS file (filename.css) is a style sheet that does not contain anything but the styles and needs to be linked. Creating an external CSS file is simple: open a blank document in Notepad, type the desired style, and click on the 'Add' button. This will create a style sheet that can be modified by changing one CSS file.

Quarter 2 AP

Lesson 1:Demand

  • Demand - kakayahan & kagustuhan g tao na makamit at bilhin ang isang produkto o serbisyo ang nagtatakda ng kanyang pangangailangan at kagustuhan
  • Batas ng Demand - pag tataas ang presyo, bababa ang dami ng gusto at kayang bilhin. Pag bababa ang presyo, tataas ang dami ng gusto at kayang bilhin
  • Ceteris Paribus - ang presyo lamang ang salik na nakaaapekto sa pagbabago ng quantity demanded habang ang ibang salik ay hindi naaapekto
  • May inverse na ugnayan sa presyo & quantity demanded dahil may substation effect na kapag tumaas ang presyo, hahanap ng mas mura ang mamimili at ang Incore effect kung saan mas marami ang halaga ng kinikita kapag mas mababa ang presyo
  • Matematikong Pormula: Demand = Kagustuhan + Kakayahan
  • Demandnd Function: Qd = a - bP
  • Demand Schedule - tsart na nagpapakita ng bilang ng mga kalakal o serbisyo na hinihiling a mga tiyak na presyo, ito rin ay nagpapakita ng mga kalakal at halaga ng mga kalakal na nais bayaran ng mga mamimili sa kanila sa presyo na iyon
  • Demand Curve - nagpapakita ng negatibong produkto relasyon ng presyo ng isang produkto at ang bilang ng produktong handing bilhin ng mamimili

Salik na nakakaapekto sa Demand:

  1. 1. Kita
  2. 2. Populasyon
  3. 3. Panlasa
  4. 4. Presyo (Complementary - dalawa ang kailangan para maging satisfied, at substitute - pamalit)
  5. 5. Inaasahan ng mga mamimili sa presyo sa hinaharap
  6. 6. Okasyon
  7. 7. Panahon o Klima

Lesson 2:Supply

  • Supply - dami ng produkto o serbisyo na gusting ibenta
  • Batas ng Supply - pag tataas ang presyo, tataas ang dammi ng gusto at kayang ibenta. Pag bababa ang presyo, bababa rin ang dami ng gusto at kayang ibenta
  • Matematikong Pormula: Supply = Pagnanais + Kakayahan
  • Supply Function: Qs = -a + bP
  • Supply Curve - paglipat sa kanan ay pagtaas ng supply, at ang paglipat sa kaliwa ay pagbaba ng supply
  • Supply Schedule - nagpapakita ng dami ng produkto na gusting gawin ng produsyer

Salik na nakakaapekto sa Suply

  1. 1. Pagbabago sa teknolohiya
  2. 2. Pagbabago sa halaga ng mga salik ng produksyon
  3. 3. Pagbabago sa bilang ng mga nagtitinda
  4. 4. Pagbabago sa presyo ng kaugnay na produkto (Complementary at Substitute Products)
  5. 5. Ekspektasyon ng presyo
  6. 6. Okasyon o Tradisyon
  7. 7. Panahon o Klima
  8. 8. Kalamidad
  9. 9. Dami ng pagkasira o pagbulok ng isang produkto
  10. 10. Tulong mula a pamahalaan o subsidiya

Lesson 3:Price Elasticity

  • Elastisidad ng Demand at Supply
  • Elastisidad - pamamaraan upang masukat ang pagtugon ng mga mamimili at nagtitinda sa pagbabago ng presyo
  • Elastisidad ng Demand (Ed) - pagtugon ng mamimili
  • Elastisidad ng Supply (Es) - pagtugon ng nagtitinda

Mga Uri

  1. Elastic (greater than 1) - mas Malaki ang nagging bahagdan ng pagtugon ng Qd o Qs kaysa sa bahagdan ng pagbabago ng presyo, produktong maraming kapalit o kahalili
  2. Di-Elastic (less than 1) - mas maliit ang nagging bahagdan ng pagtugon ng Qd o Qs kaysa sa bahagdan ng pagbabago ng presyo, produktong halos walang kapalit o kahalili
  3. Unitary (equal to 1) - pareho ang bahagdan sa presyo sa bahagdan ng pagbabago ng Qd o Qs
  4. Ganap na Elastic (∞) - pagbabago sa presyo ay madudulot ng infinite na pagbabago sa Qd o Qs
  5. Ganap na Di-Elastic (0) - Qd o Qs ay hindi tumutugon a pagbabago ng presyo

Lesson 4:Market Equilibrium

  • Equilibrium - supply at demand ay equal
  • Disequilibrium - supply at demant ay hindi equal
  • Equilibrium Price - pinagkasunduang presyo
  • Equilibrium Quantity - pinagkasunduang dami ng produkto
  • Surplus - mataas ang presyo, mababa ang demand, at mataas ang supply
  • Shortage - mababa ang presyo, mataas ang demand, at mababa ang supply

Lesson 5:Estraktura ng Pamilihan

  • Pamilihan - lugar kung saan makakamit ang konsyumer ang kanyang pangangailangan at gusto
  • Uri: Lokal, Panrehiyon, Pambansa, Pandaigdigan
  • Estraktura: Ganap na Kompetisyon - Maraming konsyumer at prodyuser, price takers, walang kontrol sa presyo, at Di-Ganap na Kompetisyon - Indibidwal na may kontrol sa presyo

Di-Ganap na Kompetisyon

  • Monopolyo - iisang prodyuser, walang kapalit, price maker
  • Oligopoly – Ilang prodyuser, may kontrol sa presyo, advertising intensive
  • Monopolistic Competition – Katamtamang dami ng prodyuser, may advertising, may kaunting kontrol sa presyo
  • Monopsony – Iisang konsyumer, maraming prodyuser, may impluwensya sa presyo

Lesson 6:Bahaging Ginagampanan ng Pamahalaan sa Regulasyon ng mga Gawaing Pangkabuhayan

  • Price control - paglimit ng produsyer, nakabase sa Republic Act 7581 o Price Control Act kung saan ito ay ang pagkontrol sa presyo ng mga bilihin
  • National Price Coordination Council - mabantayan ang presyo
  • Price Ceiling - Pinakamataas na presyo
  • Price Freeze - ipinagbabawal ang pagtaas ng presyo ng pangunahing bilihin o serbisyo
